## 📖 אודות הפרויקט

**בוטדי** הוא בוט חכם אינטראקטיבי שנועד לעזור לתלמידים והורים להכיר את חטיבת הביניים ע"ש טדי קולק בפסגת זאב. הבוט מספק מידע על בית הספר, מסלולי הלימוד, פעילויות, וכל מה שצריך לדעת על החיים בחטיבה.

### ✨ תכונות עיקריות

- 💬 **שיחה אינטראקטיבית** - שאל כל שאלה וקבל תשובה חכמה
- 📅 **מידע על יום לימודים** - שעות, מערכת, ופעילויות
- 🤝 **חיים חברתיים** - פעילויות, טיולים, ואירועים
- 🚀 **חדשנות** - מסלולים מיוחדים ותכניות ייחודיות
- 💬 **תמיכה** - מי עוזר כשיש קושי
- ⭐ **מסלולים מיוחדים** - ממר"ם, גשרים, ועוד
- 📍 **מיקום ופרטים** - איפה נמצא בית הספר
- 🎓 **מרכז למידה** - מה זה ואיך זה עובד
- 🏔️ **טיולים** - אילו טיולים יש
- 📚 **תקנון** - מה חשוב לדעת

## 🛠️ טכנולוגיות

- **HTML5** - מבנה הדף
- **CSS3** - עיצוב מודרני ורספונסיבי
- **JavaScript** - לוגיקה אינטראקטיבית
- **עברית RTL** - תמיכה מלאה בעברית וכיוון מימין לשמאל

## 🚀 התקנה והרצה

### דרישות מקדימות
- דפדפן מודרני (Chrome, Firefox, Edge, Safari)
- שרת מקומי (אופציונלי) - ניתן לפתוח ישירות את `index.html`

### הרצה מקומית

1. שכפל את הפרויקט:
```bash
git clone https://github.com/yanivmizrachiy/botteddy.git
cd botteddy
```

2. פתח את `index.html` בדפדפן או הפעל שרת מקומי:
```bash
# עם Python
python -m http.server 8000

# עם Node.js (http-server)
npx http-server

# עם PHP
php -S localhost:8000
```

3. פתח בדפדפן: `http://localhost:8000`
